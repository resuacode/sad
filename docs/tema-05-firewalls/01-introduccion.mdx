# Tema 5: Firewalls

## Introducción

Los firewalls son dispositivos o software que controlan el tráfico de red entre diferentes zonas de seguridad, permitiendo o bloqueando paquetes de datos según un conjunto de reglas de seguridad.

## Tipos de firewalls

### Firewalls de red
- Se sitúan entre la red interna y externa.
- Filtran el tráfico basado en direcciones IP, puertos y protocolos.

### Firewalls de aplicación
- Se centran en el tráfico de aplicaciones específicas.
- Analizan el contenido de las solicitudes y respuestas.

### Firewalls de próxima generación (NGFW)
- Combinan funciones de firewall tradicional con inspección profunda de paquetes (DPI).
- Pueden identificar y bloquear amenazas avanzadas.
