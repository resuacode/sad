# Autenticación Multifactor (MFA)

## ¿Qué es MFA?

La autenticación multifactor es un método de seguridad que requiere dos o más factores de verificación independientes.

## Tipos de MFA

### Two-Factor Authentication (2FA)
Utiliza exactamente dos factores de autenticación.

### Multi-Factor Authentication (MFA)
Puede utilizar dos o más factores.

## Implementaciones comunes

- **SMS**: Código enviado por mensaje de texto
- **TOTP**: Time-based One-Time Password (Google Authenticator, Authy)
- **Push notifications**: Notificaciones en dispositivos móviles
- **Hardware tokens**: Dispositivos físicos como YubiKey
- **Biometría**: Huella dactilar, reconocimiento facial

## Ventajas y desventajas

### Ventajas
- Mayor seguridad
- Protección contra phishing
- Cumplimiento normativo

### Desventajas
- Mayor complejidad para el usuario
- Posibles problemas de usabilidad
- Coste adicional
