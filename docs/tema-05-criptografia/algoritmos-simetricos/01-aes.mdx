# AES - Advanced Encryption Standard

## ¿Qué es AES?

AES es el estándar de cifrado avanzado adoptado por el gobierno estadounidense y ampliamente utilizado en todo el mundo.

## Características principales

- **Algoritmo**: Rijndael
- **Tamaños de clave**: 128, 192, 256 bits
- **Tamaño de bloque**: 128 bits
- **Rondas**: 10, 12, 14 (según el tamaño de clave)

## Modos de operación

### ECB (Electronic Codebook)
- Cada bloque se cifra independientemente
- **Problema**: Patrones visibles en datos repetitivos

### CBC (Cipher Block Chaining)
- Cada bloque se XOR con el anterior antes del cifrado
- **Requiere**: Vector de inicialización (IV)

### CTR (Counter)
- Convierte cifrador de bloque en cifrador de flujo
- **Ventaja**: Paralelizable

### GCM (Galois/Counter Mode)
- Combina cifrado y autenticación
- **Uso**: Muy común en TLS/SSL

## Implementación práctica

### Ejemplo en Python con PyCryptodome

```python
from Crypto.Cipher import AES
from Crypto.Random import get_random_bytes
from Crypto.Util.Padding import pad, unpad

# Generar clave y datos
key = get_random_bytes(32)  # AES-256
data = b"Mensaje secreto a cifrar"

# Cifrado
cipher = AES.new(key, AES.MODE_CBC)
iv = cipher.iv
encrypted_data = cipher.encrypt(pad(data, AES.block_size))

# Descifrado
cipher_decrypt = AES.new(key, AES.MODE_CBC, iv)
decrypted_data = unpad(cipher_decrypt.decrypt(encrypted_data), AES.block_size)

print(f"Original: {data}")
print(f"Cifrado: {encrypted_data.hex()}")
print(f"Descifrado: {decrypted_data}")
```

## Ventajas y desventajas

### ✅ Ventajas
- **Seguridad**: Altamente seguro con claves de 256 bits
- **Rendimiento**: Muy eficiente
- **Estándar**: Ampliamente adoptado
- **Hardware**: Soporte nativo en muchos procesadores

### ❌ Desventajas
- **Gestión de claves**: Necesita distribución segura de claves
- **Mismo origen**: No proporciona no repudio
- **Escalabilidad**: Problemas con muchos usuarios

## Casos de uso

- **Cifrado de discos**: BitLocker, FileVault
- **VPNs**: IPSec, OpenVPN
- **Protocolos web**: HTTPS/TLS
- **Bases de datos**: Cifrado de columnas sensibles
- **Comunicaciones**: Mensajería segura
